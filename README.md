# Portofolio Akuntansi - Saldyah Marsya

Website portofolio profesional untuk Saldyah Marsya, seorang Professional Accountant dengan pengalaman di bidang akuntansi dan perpajakan.

## Struktur Folder

```
PortoMarsya/
|-- index.html              # Halaman utama
|-- assets/
|   |-- css/
|   |   |-- style.css       # Stylesheet utama
|   |-- js/
|   |   |-- script.js       # JavaScript interaktif
|   |-- images/
|       |-- README.md       # Informasi gambar
|       |-- profile.jpg     # Foto profil (tambahkan)
|-- Portofolio_Akuntansi_SaldyahMarsya.md  # Data sumber
|-- README.md               # File ini
```

## Fitur Utama

### **Desain Responsive**
- Optimal untuk desktop, tablet, dan mobile
- Layout yang menyesuaikan dengan ukuran layar
- Navigation yang mobile-friendly dengan hamburger menu

### **Sections**
1. **Hero Section** - Pengenalan singkat dengan foto profil
2. **About Section** - Ringkasan profesional dan highlight keahlian
3. **Experience Section** - Timeline pengalaman kerja
4. **Education Section** - Pendidikan formal
5. **Certificates Section** - Sertifikasi profesional
6. **Skills Section** - Keahlian teknis dengan progress bar
7. **Contact Section** - Form kontak dan informasi

### **Fitur Interaktif**
- Smooth scrolling navigation
- Animasi scroll dengan Intersection Observer
- Typing effect untuk hero title
- Parallax effect pada hero section
- Hover effects pada cards
- Form validation dan notification
- Active navigation highlighting

### **Teknologi**
- **HTML5** - Semantic markup
- **CSS3** - Modern styling dengan Grid dan Flexbox
- **JavaScript ES6+** - Interaktivitas dan animasi
- **Font Awesome** - Icons
- **Google Fonts** - Typography (Inter)

## Cara Menggunakan

### **1. Menambahkan Foto Profil**
- Siapkan foto dengan ukuran 400x400px
- Simpan di `assets/images/` dengan nama `profile.jpg` atau `profile.png`
- Website akan otomatis menggunakan foto tersebut

### **2. Mengedit Konten**
Buka `index.html` dan edit bagian yang diperlukan:
- Informasi personal
- Pengalaman kerja
- Pendidikan
- Sertifikasi
- Keahlian
- Kontak

### **3. Kustomisasi Warna**
Edit file `assets/css/style.css`:
- Ubah gradien warna di CSS variables
- Sesuaikan tema dengan preferensi

### **4. Deployment**
Website siap di-deploy ke:
- GitHub Pages
- Netlify
- Vercel
- Hosting lainnya
