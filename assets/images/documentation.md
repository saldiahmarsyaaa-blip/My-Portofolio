# Folder Dokumentasi Pengalaman Kerja

Folder ini berisi file-file gambar dokumentasi untuk setiap pengalaman kerja yang ditampilkan di portofolio.

## Struktur File yang Diperlukan:

### PT. Biro Klasifikasi Indonesia (bki)
- `bki-1.jpeg` - Dokumentasi Coretax/administrasi perpajakan
- `bki-2.jpeg` - Dokumentasi rekonsiliasi data pajak  
- `bki-3.jpeg` - Dokumentasi pelaporan SPT Masa
- `bki-4.jpeg` - Dokumentasi pengelolaan PPH Unifikasi

### Sekretariat DPRD Sulawesi Selatan (dprd)
- `dprd-1.jpeg` - Dokumentasi sistem SIADINDA
- `dprd-2.jpeg` - Dokumentasi verifikasi dokumen keuangan
- `dprd-3.jpeg` - Dokumentasi rekonsiliasi data wajib pajak

### PT. Bank Tabungan Negara (btn)
- `BTN-01.jpeg` - Dokumentasi Pengarsipan dokumen
- `BTN-02.jpeg` - Dokumentasi Penginputan interim
- `BTN-03.jpeg` - Dokumentasi Penginputan interim
- `BTN-04.jpeg` - Dokumentasi Magang

## Cara Menggunakan:

1. **Siapkan gambar dokumentasi** untuk setiap pengalaman kerja
2. **Rename file** sesuai dengan naming convention di atas
3. **Letakkan di folder ini** (`assets/images/`)
4. **Format yang direkomendasikan:** JPG atau PNG
5. **Ukuran yang ideal:** 1200x800px atau rasio 3:2

## Catatan Penting:

- Jika file gambar tidak ditemukan, sistem akan otomatis menampilkan placeholder
- Pastikan gambar memiliki kualitas yang baik dan jelas
- Usahakan gambar menunjukkan pekerjaan yang relevan dengan deskripsi
- Boleh menambahkan lebih banyak gambar dengan numbering berurutan (bki-5.jpg, dprd-4.jpg, dll)

## Error Handling:

Sistem memiliki error handling otomatis:
- Jika gambar tidak ditemukan, akan muncul placeholder "Gambar tidak tersedia"
- Modal tetap bisa dibuka meskipun gambar tidak ada
- Navigation buttons tetap berfungsi dengan normal
