# Folder Images

Folder ini berisi file-file gambar yang digunakan dalam portofolio:

## File yang Diperlukan:

1. **profile-placeholder.jpg** - Foto profil Saldyah Marsya
   - Ukuran yang disarankan: 400x400px
   - Format: JPG atau PNG
   - Background yang disarankan: netral atau profesional

## Cara Mengganti Foto Profil:

1. Siapkan foto profil dengan ukuran minimal 400x400px
2. Simpan dengan nama `profile.jpg` atau `profile.png`
3. Letakkan di folder ini (assets/images/)
4. Update referensi di `index.html` jika nama file berbeda

## Format yang Didukung:
- JPG
- JPEG
- PNG
- WebP

## Catatan:
- Jika file gambar tidak ditemukan, sistem akan otomatis menampilkan icon placeholder
- Pastikan ukuran file tidak terlalu besar untuk optimasi loading
- Gunakan gambar dengan background yang konsisten untuk tampilan profesional
